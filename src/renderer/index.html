AIDE - DCS SIT V10

🗺️ NAVIGATION CARTE

  • CENTRÉ : la carte suit votre véhicule
  • LIBRE : panoramique manuel
  • ROTATIF : carte orientée selon votre cap
  • Clic gauche + drag : panoramique de la carte (en mode LIBRE)
  • Molette : zoom avant/arrière
  • F11 : plein écran

🖱️ SÉLECTION D'UNITÉS (RTS)

  • Clic sur une unité alliée : sélectionne le leader du groupe (autres unités en orange)
  • Clic dans le vide (court) : désélectionne
  • Ctrl + clic : ajoute/retire une unité de la sélection
  • Shift + drag : encadrement rectangulaire de sélection
  • Shift maintenu entre plusieurs encadrements : accumule les sélections

🎮 ORDRES AUX UNITÉS SÉLECTIONNÉES

  • Clic droit court : déplacer vers ce point (remplace les WPs)
  • Ctrl + clic droit court : ajoute ce WP à la chaîne existante
  • Clic droit long (>400ms) + drag : mode formation — chaque unité reçoit sa position sur une ligne orientée par la souris (espacement 50m)
  • Panneau bas : boutons vitesse / formation / ROE appliqués à tous les groupes sélectionnés
  • Bouton ✕ Route : efface les WPs et envoie Hold aux unités sélectionnées

📍 PLACEMENTS (panneau droit)

  • TGT : cible (losange blanc + tier dots)
  • DRONE : drone de recon (voir section dédiée)
  • ERBL : point de référence (distance/azimut depuis ce point)
  • DEL : clic sur un marqueur pour le supprimer

🚁 DRONE (reconnaissance)

  • Mode Joueur : le drone part de votre véhicule
  • Mode GM/Tactcom : clic sur une unité alliée (pulsant orange) pour définir la source
  • Clic carte : pose un WP (max 2)
  • 1 WP = station + rotation 360° sur place. 2 WP = patrouille A↔B
  • Entrée : valide le plan — Escape : annule
  • Cône de vision 120° avant. Détection : 100% à <1km, 66% à 1-2km, 33% à 2-3km
  • Durée de vie max 15 min. Max 3 drones (joueur) / 5 drones (GM/Tactcom)

🎯 JFO (appui feu)

  • Clic sur JFO : ouvre la liste des groupes d'artillerie
  • Sélection manuelle ou mode CARTE (unités ARTY pulsent orange)
  • Clic droit sur la carte : pose la cible et ouvre le menu munitions
  • 12 munitions disponibles : obus classiques, fumigènes (tactique/couleur), flares, mines, cluster

✏️ DRAW (dessin tactique)

  • Outils : ligne, polyligne, rectangle, cercle, ovale, texte, icônes OTAN
  • Styles de ligne : plein, double, tirets, pointillés, croix, dents, fortif, flèche
  • Mode ✥ Sélection : clic sur un dessin (tolérance 60px) pour le sélectionner
  • Delete : supprime le dessin sélectionné
  • Sauvegarde/chargement presets, export/import JSON

🛣️ PLAN DE ROUTE

  • Mode COORD : saisie manuelle des waypoints
  • Mode MAP : clics sur carte, Entrée pour valider
  • Suppression d'un plan local : supprime aussi côté serveur multi
  • Les plans d'autres clients apparaissent en superposition violette

📊 TOPO (lignes de niveau)

  • Bouton TOPO : active les contours altimétriques
  • Bouton REFALT : demande une nouvelle grille au serveur DCS
  • Curseurs verticaux hex-tech en haut-droite : DEN (densité) / OPA (opacité)
  • Chaque client a sa propre requête REFALT indépendante

💬 MESSAGES

  • Bouton MSG (panneau gauche) : ouvre la fenêtre d'envoi
  • Cocher SIT : affiche dans le chat du SIT (coin haut-gauche)
  • Cocher DCS : affiche aussi dans DCS via trigger.action.outText
  • Par défaut SIT est coché. Le choix DCS est mémorisé entre sessions
  • Filtrage par coalition respecté (rouge/bleue/toutes)

⚠️ DDM / CR (alertes)

  • DDM : détection automatique de missiles (son + marqueur)
  • CLEAR : efface les alertes DDM
  • CR : compte-rendus de tir (destructions ennemies)
  • CRCL : efface les CR affichés

👥 MULTI

  • Bouton MULTI (ruban haut) : connexion à un serveur SIT
  • Modes : Joueur / Commandant Tactique / Maître du Jeu
  • GM : voit et commande les deux coalitions, chacune avec sa couleur
  • Dessins, TGT, plans, drones sont partagés entre clients
  • Drones d'autres clients en vert, les vôtres en cyan

🚑 EVASAN / CSAR (sauvetage pilote)

  • Détection automatique des crashes / éjections / morts des pilotes joueurs de votre coalition
  • Bouton croix rouge en haut du panneau droit : pastille rouge avec nombre de pilotes abattus non traités
  • Clic sur le bouton : centre la carte sur le prochain crash. Clic répété : cycle entre les crashes
  • Clic sur la croix rouge affichée sur la carte : ouvre le menu des choix
  • Choix 1 — Ne rien faire : le pilote pourra reprendre un appareil 15 min après le crash (heure de Paris)
  • Choix 2 — EVASAN LÉGÈRE : envoie le groupe nommé MERCURE vers le crash (sur route ou tout terrain, 150 km/h)
  • Choix 3 — EVASAN LOURDE : envoie le groupe nommé CHROME vers le crash (blindé, protégé)
  • Choix 4 — EVASAN JOUEUR : fumigène bleu sur le crash + broadcast coalition DCS. Si un hélico joueur reste <50m <5m/s pendant 10s, la récupération est validée
  • Prérequis mission : pour les EVASAN LÉGÈRE/LOURDE, la mission DCS doit contenir un groupe sol nommé MERCURE (ou variantes : Mercure, mercure) et/ou CHROME (ou chrome, ochrome)
  • Si le groupe n'existe pas, un message d'alerte apparaît dans le chat SIT

⚙️ CONFIGURATION

  • Thèmes : Sombre / Clair / ICONE (tablette durcie militaire)
  • Chargement carte PNG avec calibration automatique
  • OFFSET en bas : ajustement fin de la carte
